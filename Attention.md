# Docusaurus Markdown 渲染注意事项

## 1. 段落换行问题

### 问题描述
Markdown 编辑器中的自动换行不会在渲染时显示为换行，必须使用空行来分隔段落。

### 解决方案
- 段落之间必须有一个空行
- 不要依赖编辑器的自动换行，应该手动添加空行
- 句号后应该添加空行（如果开始新段落）

**错误示例：**
```markdown
第一句话。
第二句话
```

**正确示例：**
```markdown
第一句话。

第二句话。
```

## 2. 数学公式格式

### aligned 环境格式问题
当使用 `aligned` 环境时，`$$` 符号必须独立成行，否则会导致后续内容渲染异常。

**错误示例：**
```markdown
$$\begin{aligned}
v_1 &= r_1 + \gamma r_2 + \gamma^2 r_3 + \cdots, \\
v_2 &= r_2 + \gamma r_3 + \gamma^2 r_4 + \cdots, \\
\end{aligned}$$
```

**正确示例：**
```markdown
$$
\begin{aligned}
v_1 &= r_1 + \gamma r_2 + \gamma^2 r_3 + \cdots, \\
v_2 &= r_2 + \gamma r_3 + \gamma^2 r_4 + \cdots, \\
\end{aligned}
$$
```

### 关键要点
- `$$` 开始和结束符号必须独立成行
- 公式块前后必须有空行
- 如果公式后内容挤在一起，检查 `$$` 是否独立成行

## 3. 代码块格式

### 代码示例
代码示例应该使用代码块包裹，而不是直接写在段落中。

**错误示例：**
```markdown
return_a = 0+0+0+1 = 1  
return_b = 0+(-1)+0+1 = 0
```

**正确示例：**
```markdown
```
return_a = 0+0+0+1 = 1
return_b = 0+(-1)+0+1 = 0
```
```

## 4. 图片格式

### 图片前后空行
图片前后必须有空行，否则可能影响渲染。

**错误示例：**
```markdown
这是文字
![Image](/img/rl/image.png)
这是更多文字
```

**正确示例：**
```markdown
这是文字

![Image](/img/rl/image.png)

这是更多文字
```

## 5. 标题和内容之间

### 空行分隔
标题和内容之间必须有空行。

**错误示例：**
```markdown
### 1.1 State and Action
状态是描述智能体相对于环境所处状态的核心概念。
```

**正确示例：**
```markdown
### 1.1 State and Action

状态是描述智能体相对于环境所处状态的核心概念。
```

## 6. 长段落分段

### 可读性
长段落应该适当分段，提高可读性。

**建议：**
- 每个段落不超过 3-4 句话
- 使用空行分隔不同主题的内容
- Summary 部分使用小标题（`####`）来分段

## 7. 列表格式

### 列表前后空行
列表前后必须有空行。

**错误示例：**
```markdown
这是文字
- 列表项1
- 列表项2
这是更多文字
```

**正确示例：**
```markdown
这是文字

- 列表项1
- 列表项2

这是更多文字
```

## 8. 检查清单

在提交文档前，检查以下事项：

- [ ] 所有段落之间都有空行
- [ ] 所有 `aligned` 环境的 `$$` 符号独立成行
- [ ] 所有图片前后都有空行
- [ ] 所有标题和内容之间都有空行
- [ ] 代码示例使用代码块包裹
- [ ] 长段落适当分段
- [ ] 句号后适当添加空行（如果开始新段落）

## 9. 常见问题排查

### 问题：内容挤在一起
**可能原因：**
1. 段落之间缺少空行
2. `aligned` 环境的 `$$` 没有独立成行
3. 图片前后缺少空行

### 问题：数学公式渲染异常
**可能原因：**
1. `$$` 符号没有独立成行
2. 公式块前后缺少空行
3. LaTeX 语法错误

### 问题：代码不显示
**可能原因：**
1. 代码块没有使用正确的反引号（三个反引号）
2. 代码块前后缺少空行
