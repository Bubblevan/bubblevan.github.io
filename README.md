## 项目简介

- **我** - 个人简历和基本信息
- **本科笔记** - 生物医学工程专业学习笔记
- **项目介绍** - 项目经历和技术栈
- **科研经历** - 科研项目和学术成果
- **自学笔记** - 技术学习和心得

## 本地开发

### 环境要求
- Node.js >= 18.0
- npm 或 yarn

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run start
```

### 构建生产版本
```bash
npm run build
```
