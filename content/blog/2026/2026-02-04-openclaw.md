```
# 1. 删除旧版本（如果之前用 apt 安装的）
sudo apt remove nodejs npm -y

# 2. 添加 NodeSource 仓库（包含 v22）
curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -

# 3. 安装 Node.js
sudo apt install -y nodejs

# 4. 验证版本
node --version
```
这里比较麻烦，因为对node的要求是22，而且需要wsl，这里索性重装一遍。
> 不得不提，wsl对于Windows挂载的`/mnt/d`访问真的很慢。

```
# 1. 重新安装 pnpm（建议）
sudo npm install -g pnpm

# 2. 清除之前的安装缓存（如果之前有错误）
rm -rf node_modules
rm -f pnpm-lock.yaml

# 3. 重新安装依赖
pnpm install

# 4. 构建项目
pnpm ui:build # auto-installs UI deps on first run
pnpm build

pnpm openclaw onboard --install-daemon

# Dev loop (auto-reload on TS changes)
pnpm gateway:watch
```
