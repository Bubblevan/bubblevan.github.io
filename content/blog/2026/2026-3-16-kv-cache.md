## KV Cache 的引入
Transformer 模型的核心是自注意力（Self-Attention）。假设我们有一个长度为T的输入序列（每个位置是一个token的向量表示），那么：
- 对每个位置的向量，乘上3个不同的矩阵Q、K、V，分别得到查询（Query）、键（Key）、值（Value）
- 然后对于第i个位置，用自己的Query去和所有位置（包括自己）的可以做点积，得到注意力分数（除以sqrt(d_k)缩放），再经过Softmax得到权重。
- 用这些权重去加权求和所有位置的Value，得到第i个位置的输出。

\[
\text{Attention}(Q,K,V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V
\]

当我们做文本生成时，我们用的是自回归解码——每次只产生下一个token，然后把新token拼到已有的序列后面，再做为输入去预测再下一个token。
前面我们提到，Key和Value的计算是通过该token的输入向量乘上K和V的权重矩阵得到的，而**推理**的时候权重矩阵是不变的。
这涉及到一个**因果编码**的概念，每个token计算自注意力时只能看到左边的token，看不到右边的token（Softmax后为0）。在Transformer的每一层，所有token的输入是通手机进入的，但因果编码确保了无论第N+1个token是否存在，是什么值，它们在第L层的输出都只由L-1层中他们自己及左边的token决定，而这些左边的token在第L-1层的输出在之前计算时就已经固定了。

因此，在第一次计算整个初始序列（Prompt）时，我们把每一层、每个头计算出的Key和Value矩阵保存下来（每个新token的Query都会变这里不保存）；每次只对新token计算它的Query、Key、Value；然后把这心拼接到之前保存的Key和Value后面（沿着序列长度方向拼接），得到完整的Key矩阵和Value矩阵；然后对于新新token的注意力计算，只需要用它的Query和这个完整的Key矩阵做点积（再Softmax），再对完整的Value矩阵做加权和，就能得到新新token的输出。

假设我们有一个极简的 Transformer，只有一层自注意力。初始 prompt 是 `["今天", "天气"]`（两个 token）。

- **Prefill 阶段**：
  - 所谓Prefill意为把整个Prompt一次性输入模型，计算所有层的KV并缓存下来，同时计算出最后一个位置的logits，从而得到第一个生成的token（这个阶段是并行的，因为所有token可以同时参与attention的计算）
  - 输入这两个 token，经过 embedding 得到向量 \(x_1, x_2\)。在自注意力层：
  - 计算所有 token 的 \(Q, K, V\)（假设用矩阵乘法 \(Q = XW^Q\) 等，但为了直观，我们只看单个 token）。
  - 对于第一个 token（位置 1），它只能看到自己，所以它的输出 \(y_1 = \text{Attention}(Q_1, K_{[1]}, V_{[1]})\)。
  - 对于第二个 token（位置 2），它能看到自己和第一个 token，所以它的输出 \(y_2 = \text{Attention}(Q_2, K_{[1,2]}, V_{[1,2]})\)。
  - 在计算过程中，我们得到了 \(K_1, V_1, K_2, V_2\) 并缓存下来。

现在我们要生成第三个 token，假设上一步预测出第三个 token 是 "很"。

- **Decode 阶段（生成第三个 token）**：
  - 每步只输入一个新token（通常是上一步生成的token），计算它自己的Q、K、V，然后拼接KV，再用Q和拼接后的完整K做attention，得到该token经过每一层的输出，最后预测下一个token（这个阶段是串行的，必须等上一步token生成后才能进行下一步）
  - 输入只有第三个 token "很"，它的向量是 \(x_3\)（来自 embedding 或上一层输出）。在自注意力层：
  - 计算 \(Q_3, K_3, V_3\)。
  - 现在我们有缓存的 \(K_1, V_1, K_2, V_2\)，把它们和 \(K_3, V_3\) 拼起来，得到完整的 \(K_{[1,2,3]}\) 和 \(V_{[1,2,3]}\)。
  - 对于第三个 token，它需要看到所有前面的 token，所以它的输出 \(y_3 = \text{Attention}(Q_3, K_{[1,2,3]}, V_{[1,2,3]})\)。

注意：在这个过程中，我们完全没有重新计算 \(K_1, V_1, K_2, V_2\)，因为它们和之前 prefill 阶段算出的值**完全相同**。因为：
- \(K_1\) 只依赖于 \(x_1\)（以及固定的权重矩阵），而 \(x_1\) 和权重矩阵都没变。
- 同样，\(K_2\) 依赖于 \(x_2\)，而 \(x_2\) 也没变。
- 为什么 \(x_1\) 和 \(x_2\) 没变？因为 \(x_1\) 是 embedding 层直接查表得到的（或者来自上一层），而 embedding 层是固定的；\(x_2\) 在 prefill 阶段已经确定，且后续没有新的 token 影响它（因果性保证）。

如此一来模型的计算量就不会随着历史长度增长而增加计算量（但内存占用缓存KV会越来越多）。
在Huggingface的Transformer库中，KV Cache就是`past_key_values`这个元祖，长度等于层数，每一层对应一个`(key, value)`，其中`key`、`value`的形状通常为`(batch_size, num_heads, seq_len, head_dim)`。
- `seq_len`是到目前为止已经缓存的序列长度
- 每次decode新token，新value和key都沿着这个维度拼接上去
- encoder-decoder模型里会缓存cross-attention的KV，但纯decoder模型通常只有self-attention的

## VLM KV Cache
有 KV cache 时，老历史不重算，只追加新 token 的那一小段。于是 decode 越长，收益通常越明显。
然而，对于多模态KV Cache来说，其**优化的是decode**，但**并非所有延迟都来自decode**。
在 VLM 里，图像本身要先经过视觉编码、投影、多模态拼接，这些成本很多都**发生在 prefill**，所以预期会看到：
- 总时延下降了，但没有想象中那么大
- 输出越长，cache 越值钱
- 图像越多，prefill 越重，cache 的“相对收益”会被稀释
因此，这里又涉及了两类Cache的方法：
### `used_cache=True`
在**一次生成**里面，第2、3、4个token复用前面token的KV
### prefix caching
处理的是跨请求，如果两个请求共享完全一样的前缀，就尝试复用这段前缀的缓存。
> vLLM里有一个Automatic Prefix Caching，他会把Image hash编进缓存key，不会只看`<image>`占位符，避免文本一样当图片不同时误用

- **同图 + 同文本前缀**：更有机会复用
- **同文本 + 换图**：通常不能直接复用同一份多模态前缀缓存（这也是VLA里常见的情况吧，文本模态超级稀疏）

## 指标
| 类别         | 指标                        | 描述                                                                 |
|--------------|-----------------------------|----------------------------------------------------------------------|
| **延迟类**   | prefill latency             | 整段 prompt 进入模型，到拿到 first-token logits 的时间               |
|              | TTFT (time to first token)  | 在“手写 greedy decode”实验中可近似看作 prefill latency               |
|              | decode latency              | 除第一个 token 外，后续 token 的生成时间                             |
|              | end-to-end latency          | 从送入请求到生成结束                                                 |
| **吞吐类**   | decode tokens/s             | 后续生成 token 的速度                                                |
|              | end-to-end tokens/s         | 输出 token 数 / 总耗时                                               |
| **资源类**   | peak GPU memory allocated   |                                                                      |
|              | peak GPU memory reserved    |                                                                      |
|              | 进程 RSS（CPU 内存）         |                                                                      |
| **质量一致性** | 开关 cache 前后，输出文本是否一致 |                                                                      |
|              | JSON / 格式化输出是否稳定     |                                                                      |
|              | 是否出现答非所问             |                                                                      |

### 第1阶段基线
- 跑`model.generate()`
- 对比`use_cache=False`与`True`
- 记录总时延、输出tokens数、tokens/s、GPU peak、CPU RSS
```python
import argparse
import json
import os
import statistics
import time
from dataclasses import dataclass, asdict
from typing import Any, Dict, List

import psutil
import torch
from PIL import Image
from transformers import AutoProcessor, Qwen2_5_VLForConditionalGeneration

try:
    from qwen_vl_utils import process_vision_info
except ImportError as e:
    raise SystemExit(
        "Missing qwen-vl-utils. Install it with: pip install qwen-vl-utils"
    ) from e


@dataclass
class RunResult:
    case_name: str
    use_cache: bool
    run_idx: int
    elapsed_s: float
    output_tokens: int
    output_toks_per_s: float
    peak_gpu_alloc_mb: float
    peak_gpu_reserved_mb: float
    rss_mb: float
    output_text: str


def to_device(batch: Dict[str, Any], device: torch.device) -> Dict[str, Any]:
    out = {}
    for k, v in batch.items():
        if torch.is_tensor(v):
            out[k] = v.to(device)
        else:
            out[k] = v
    return out


def build_inputs(processor, messages: List[Dict[str, Any]], device: torch.device):
    text = processor.apply_chat_template(
        messages,
        tokenize=False,
        add_generation_prompt=True,
    )
    image_inputs, video_inputs = process_vision_info(messages)
    inputs = processor(
        text=[text],
        images=image_inputs,
        videos=video_inputs,
        padding=True,
        return_tensors="pt",
    )
    return to_device(inputs, device)


def reset_cuda_stats():
    if torch.cuda.is_available():
        torch.cuda.empty_cache()
        torch.cuda.reset_peak_memory_stats()
        torch.cuda.synchronize()


def gpu_stats_mb():
    if not torch.cuda.is_available():
        return 0.0, 0.0
    alloc = torch.cuda.max_memory_allocated() / 1024**2
    reserved = torch.cuda.max_memory_reserved() / 1024**2
    return alloc, reserved


def run_once(
    model,
    processor,
    device: torch.device,
    case: Dict[str, Any],
    use_cache: bool,
    run_idx: int,
) -> RunResult:
    inputs = build_inputs(processor, case["messages"], device)

    max_new_tokens = int(case["max_new_tokens"])
    process = psutil.Process(os.getpid())

    reset_cuda_stats()
    rss_before = process.memory_info().rss / 1024**2

    with torch.inference_mode():
        t0 = time.perf_counter()
        output_ids = model.generate(
            **inputs,
            do_sample=False,
            max_new_tokens=max_new_tokens,
            use_cache=use_cache,
        )
        if torch.cuda.is_available():
            torch.cuda.synchronize()
        elapsed = time.perf_counter() - t0

    input_len = inputs["input_ids"].shape[1]
    generated_ids = output_ids[:, input_len:]
    output_tokens = int(generated_ids.shape[1])

    output_text = processor.batch_decode(
        generated_ids,
        skip_special_tokens=True,
        clean_up_tokenization_spaces=False,
    )[0]

    peak_alloc_mb, peak_reserved_mb = gpu_stats_mb()
    rss_after = process.memory_info().rss / 1024**2
    rss_mb = max(rss_before, rss_after)

    return RunResult(
        case_name=case["name"],
        use_cache=use_cache,
        run_idx=run_idx,
        elapsed_s=elapsed,
        output_tokens=output_tokens,
        output_toks_per_s=(output_tokens / elapsed) if elapsed > 0 and output_tokens > 0 else 0.0,
        peak_gpu_alloc_mb=peak_alloc_mb,
        peak_gpu_reserved_mb=peak_reserved_mb,
        rss_mb=rss_mb,
        output_text=output_text,
    )


def summarize(results: List[RunResult]):
    grouped = {}
    for r in results:
        grouped.setdefault((r.case_name, r.use_cache), []).append(r)

    print("\n========== SUMMARY ==========")
    for (case_name, use_cache), rows in grouped.items():
        elapsed = [x.elapsed_s for x in rows]
        toksps = [x.output_toks_per_s for x in rows]
        alloc = [x.peak_gpu_alloc_mb for x in rows]
        reserved = [x.peak_gpu_reserved_mb for x in rows]
        print(f"[{case_name}] use_cache={use_cache}")
        print(f"  elapsed_s(mean):      {statistics.mean(elapsed):.4f}")
        print(f"  output_toks_per_s:    {statistics.mean(toksps):.4f}")
        print(f"  peak_alloc_mb(mean):  {statistics.mean(alloc):.2f}")
        print(f"  peak_resv_mb(mean):   {statistics.mean(reserved):.2f}")

    print("\n====== OUTPUT CONSISTENCY CHECK ======")
    by_case = {}
    for r in results:
        by_case.setdefault(r.case_name, {}).setdefault(r.use_cache, []).append(r.output_text)

    for case_name, bucket in by_case.items():
        if True in bucket and False in bucket:
            text_true = bucket[True][0]
            text_false = bucket[False][0]
            same = text_true == text_false
            print(f"[{case_name}] cache_on_vs_off_same_output = {same}")


def main():
    parser = argparse.ArgumentParser()
    parser.add_argument("--model-path", type=str, required=True)
    parser.add_argument("--cases-json", type=str, required=True)
    parser.add_argument("--dtype", type=str, default="bfloat16", choices=["float16", "bfloat16", "float32"])
    parser.add_argument("--attn-implementation", type=str, default="sdpa")
    parser.add_argument("--warmup", type=int, default=1)
    parser.add_argument("--repeat", type=int, default=3)
    parser.add_argument("--save-jsonl", type=str, default="results_generate.jsonl")
    args = parser.parse_args()

    if args.dtype == "bfloat16":
        dtype = torch.bfloat16
    elif args.dtype == "float16":
        dtype = torch.float16
    else:
        dtype = torch.float32

    device = torch.device("cuda" if torch.cuda.is_available() else "cpu")

    if not os.path.isdir(args.model_path):
        raise SystemExit(f"Error: Model path '{args.model_path}' is not a directory.")

    model = Qwen2_5_VLForConditionalGeneration.from_pretrained(
        args.model_path,
        torch_dtype=dtype,
        attn_implementation=args.attn_implementation,
        device_map="auto" if torch.cuda.is_available() else None,
        local_files_only=True,
    )
    if not torch.cuda.is_available():
        model.to(device)
    model.eval()

    processor = AutoProcessor.from_pretrained(args.model_path)

    with open(args.cases_json, "r", encoding="utf-8") as f:
        cases = json.load(f)

    all_results: List[RunResult] = []

    for case in cases:
        print(f"\n===== CASE: {case['name']} =====")
        for use_cache in [False, True]:
            for i in range(args.warmup):
                _ = run_once(model, processor, device, case, use_cache=use_cache, run_idx=-(i + 1))

            for i in range(args.repeat):
                result = run_once(model, processor, device, case, use_cache=use_cache, run_idx=i)
                all_results.append(result)
                print(
                    f"use_cache={use_cache} run={i} "
                    f"elapsed={result.elapsed_s:.4f}s "
                    f"out_tokens={result.output_tokens} "
                    f"toks/s={result.output_toks_per_s:.4f} "
                    f"peak_alloc={result.peak_gpu_alloc_mb:.1f}MB"
                )

    with open(args.save_jsonl, "w", encoding="utf-8") as f:
        for r in all_results:
            f.write(json.dumps(asdict(r), ensure_ascii=False) + "\n")

    summarize(all_results)
    print(f"\nSaved raw results to: {args.save_jsonl}")


if __name__ == "__main__":
    main()
```

### 第2阶段手搓`past_key_values`
用同一个 Qwen2.5-VL 模型，但先跑“文本模式”来观察 cache 机制本身。这么做是为了把“KV cache 原理”从“图像输入工程细节”里拆出来。
这里把时间拆成Prefill、TTFT（近似看作Prefill）、decode、Total；
并尽量打印第一层cache的形状看看到底长什么样
```python
import argparse
import time
from typing import Any, Dict, Optional, Tuple

import torch
from transformers import AutoProcessor, Qwen2_5_VLForConditionalGeneration


def tensor_to_device(batch: Dict[str, Any], device: torch.device) -> Dict[str, Any]:
    out = {}
    for k, v in batch.items():
        if torch.is_tensor(v):
            out[k] = v.to(device)
        else:
            out[k] = v
    return out


def inspect_cache_shape(past_key_values) -> Tuple[Optional[Tuple[int, ...]], Optional[Tuple[int, ...]]]:
    try:
        if hasattr(past_key_values, "key_cache") and len(past_key_values.key_cache) > 0:
            k = tuple(past_key_values.key_cache[0].shape)
            v = tuple(past_key_values.value_cache[0].shape)
            return k, v
    except Exception:
        pass
    return None, None


def main():
    parser = argparse.ArgumentParser()
    parser.add_argument("--model-path", type=str, required=True)
    parser.add_argument("--prompt", type=str, required=True)
    parser.add_argument("--max-new-tokens", type=int, default=64)
    parser.add_argument("--dtype", type=str, default="bfloat16", choices=["float16", "bfloat16", "float32"])
    parser.add_argument("--attn-implementation", type=str, default="sdpa")
    args = parser.parse_args()

    if args.dtype == "bfloat16":
        dtype = torch.bfloat16
    elif args.dtype == "float16":
        dtype = torch.float16
    else:
        dtype = torch.float32

    device = torch.device("cuda" if torch.cuda.is_available() else "cpu")

    model = Qwen2_5_VLForConditionalGeneration.from_pretrained(
        args.model_path,
        torch_dtype=dtype,
        attn_implementation=args.attn_implementation,
        device_map="auto" if torch.cuda.is_available() else None,
    )
    if not torch.cuda.is_available():
        model.to(device)
    model.eval()

    processor = AutoProcessor.from_pretrained(args.model_path)

    messages = [
        {
            "role": "user",
            "content": [
                {"type": "text", "text": args.prompt}
            ],
        }
    ]

    # 这里用 text-only 模式，专注观察 cache 机制
    enc = processor.apply_chat_template(
        messages,
        tokenize=True,
        add_generation_prompt=True,
        return_dict=True,
        return_tensors="pt",
    )
    enc = tensor_to_device(enc, device)

    if torch.cuda.is_available():
        torch.cuda.empty_cache()
        torch.cuda.reset_peak_memory_stats()
        torch.cuda.synchronize()

    with torch.inference_mode():
        # ===== 1) prefill =====
        t0 = time.perf_counter()
        outputs = model(
            **enc,
            use_cache=True,
            return_dict=True,
        )
        if torch.cuda.is_available():
            torch.cuda.synchronize()
        prefill_s = time.perf_counter() - t0

        past_key_values = outputs.past_key_values
        first_token = outputs.logits[:, -1, :].argmax(dim=-1, keepdim=True)

        # TTFT 在这个脚本里近似看作 prefill 完成的时刻
        ttft_s = prefill_s

        k_shape, v_shape = inspect_cache_shape(past_key_values)
        print("First-layer cache shapes:")
        print("  key:", k_shape)
        print("  val:", v_shape)

        generated = [first_token]
        next_token = first_token

        processed_len = enc["input_ids"].shape[1]
        attention_mask = enc["attention_mask"]

        # ===== 2) decode loop =====
        t1 = time.perf_counter()
        for _ in range(args.max_new_tokens - 1):
            attention_mask = torch.cat(
                [
                    attention_mask,
                    torch.ones(
                        (attention_mask.shape[0], 1),
                        dtype=attention_mask.dtype,
                        device=attention_mask.device,
                    ),
                ],
                dim=1,
            )

            cache_position = torch.arange(
                processed_len,
                processed_len + 1,
                device=device,
            )

            outputs = model(
                input_ids=next_token,
                attention_mask=attention_mask,
                past_key_values=past_key_values,
                use_cache=True,
                cache_position=cache_position,
                return_dict=True,
            )

            past_key_values = outputs.past_key_values
            next_token = outputs.logits[:, -1, :].argmax(dim=-1, keepdim=True)
            generated.append(next_token)
            processed_len += 1

            eos_id = processor.tokenizer.eos_token_id
            if eos_id is not None and int(next_token.item()) == int(eos_id):
                break

        if torch.cuda.is_available():
            torch.cuda.synchronize()
        decode_s = time.perf_counter() - t1

    gen_ids = torch.cat(generated, dim=1)
    text = processor.tokenizer.decode(
        gen_ids[0],
        skip_special_tokens=True,
        clean_up_tokenization_spaces=False,
    )

    total_s = prefill_s + decode_s
    new_tokens = int(gen_ids.shape[1])

    peak_alloc_mb = 0.0
    peak_reserved_mb = 0.0
    if torch.cuda.is_available():
        peak_alloc_mb = torch.cuda.max_memory_allocated() / 1024**2
        peak_reserved_mb = torch.cuda.max_memory_reserved() / 1024**2

    print("\n========== METRICS ==========")
    print(f"prefill_s           = {prefill_s:.6f}")
    print(f"ttft_s              = {ttft_s:.6f}")
    print(f"decode_s            = {decode_s:.6f}")
    print(f"total_s             = {total_s:.6f}")
    print(f"new_tokens          = {new_tokens}")
    print(f"decode_tokens_per_s = {(max(new_tokens - 1, 1) / decode_s) if decode_s > 0 else 0.0:.6f}")
    print(f"total_tokens_per_s  = {(new_tokens / total_s) if total_s > 0 else 0.0:.6f}")
    print(f"peak_alloc_mb       = {peak_alloc_mb:.2f}")
    print(f"peak_reserved_mb    = {peak_reserved_mb:.2f}")

    print("\n========== GENERATED ==========")
    print(text)


if __name__ == "__main__":
    main()
```
