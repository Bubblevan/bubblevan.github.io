## uv 安装
```bash
python -m pip install --upgrade uv

uv venv --python 3.12 --seed
source .venv/bin/activate

uv pip install vllm --torch-backend=auto
```

最后来启动这个vllm：
```bash
vllm serve /root/backup/latency/Qwen2.5-VL-3B-Instruct \
  --dtype auto \
  --max-model-len 4096 \
  --gpu-memory-utilization 0.45 \
  --limit-mm-per-prompt '{"image":4,"video":0}'
```
**Qwen2.5-VL-3B 成功被 vLLM 识别并加载，VLM 路由可用，prefix caching 已开启，KV cache 已分配完成，服务端已经可以收 `/v1/chat/completions` 请求了。**
