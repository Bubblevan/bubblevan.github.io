还有kubectl指令：
```bash
# 1. 查看Pod 状态
kubectl get pods -n zheda-agent

# 2. 查看 Service 和 Pod 的简要状态
kubectl get svc,pods -n zheda-agent | grep -E "(query|api-gateway|STATUS)"

# 3. 通过 label 查看 query-service 日志（实时，最近100行）
kubectl logs -n zheda-agent -l app=query-service -f --tail=100

# 4. 或者先获取具体 Pod 名称再查看日志
POD_NAME=$(kubectl get pods -n zheda-agent -l app=query-service -o jsonpath='{.items[0].metadata.name}')
kubectl logs -n zheda-agent $POD_NAME -f --tail=100

# 5. 查看 api-gateway 日志（最近50行）
kubectl logs -n zheda-agent -l app=api-gateway -f --tail=50

# 6. 查看最近20条事件（按时间排序）
kubectl get events -n zheda-agent --sort-by='.lastTimestamp' | tail -20
```
