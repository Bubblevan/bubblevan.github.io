## 今日总结
### Stablepay
```bash
blocked plugin candidate: world-writable path (/mnt/d/mylab/stablepay/stablepay-openclaw-plugin, mode=777)
```
在 WSL（Linux 子系统） 里运行，Windows 挂载的目录权限是 777（任何人都能读写执行），OpenClaw 出于安全考虑，直接禁止加载这个插件，所以需要改成 755（仅所有者可写），让 OpenClaw 允许加载插件。
`chmod -R 755 /mnt/d/mylab/stablepay/stablepay-openclaw-plugin`这个命令其实做不到，只能`cp -r /mnt/d/mylab/stablepay/stablepay-openclaw-plugin ~/stablepay-openclaw-plugin`
```bash
# 复制到 home 目录（Linux 原生 ext4，chmod 有效）
cp -r /mnt/d/mylab/stablepay/stablepay-openclaw-plugin ~/stablepay-openclaw-plugin

# 修正权限
chmod -R 755 ~/stablepay-openclaw-plugin
chmod 644 ~/stablepay-openclaw-plugin/src/*.ts
chmod 644 ~/stablepay-openclaw-plugin/package.json

# 重新安装（从 Linux 路径）
openclaw plugins install --link ~/stablepay-openclaw-plugin

# 重启网关
openclaw gateway restart

# 开启对话
openclaw tui
```

安装 **openwallet**：`curl -fsSL https://docs.openwallet.sh/install.sh | bash`
