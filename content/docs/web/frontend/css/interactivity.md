### **第四部分：视觉效果与交互性**

1.  **CSS 变量 (Variables): 让样式可复用**
    *   定义 (`--main-color: #...`) 与使用 (`var(--main-color)`)。
    *   全局与局部作用域。

2.  **过渡 (Transitions): 平滑的状态变化**
    *   `transition` 属性：`property`, `duration`, `timing-function`, `delay`。

3.  **动画 (Animations): 让页面动起来**
    *   `@keyframes` 规则：定义动画的关键帧。
    *   `animation` 属性：绑定动画到元素。
