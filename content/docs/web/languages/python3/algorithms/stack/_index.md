栈、队列、双端队列和列表都是数据集合的例子，它们的元素顺序取决于添加或移除的方式。一旦一个元素被添加，它就会相对于在它之前和之后的其他元素保持在那个位置。诸如此类的集合通常被称为线性数据结构。

线性结构可以被认为有两个端点。有时这些端点被称为“左”和“右”，在某些情况下也被称为“前”和“后”。
> 你也可以称它们为“上”和“下”。

给这些端点起的名称并不重要。区分一种线性结构与另一种线性结构的是元素的添加和移除方式，特别是这些添加和移除操作发生的位置。例如，一种结构可能只允许在一端添加新元素。有些结构可能允许从两端移除元素。
