从 C 转向 Python，就像是从C1换到了C2。C 语言要求关注内存、指针和类型声明；而 Python 更关注“你想做什么”，所以拿来刷Leetcode是最好的语言——JD里面对语言有明确要求的除外（C++/Go/Java）

在 C 语言里，打印一句话需要包含头文件、写 `main` 函数、加大括号。而在 Python 里，只需要一行：

```python
print("Hello, World!")

```

没有分号，没有复杂的入口，这就是 Python 的风格。

## 一、 字符串

在 C 语言中，字符串是 `char` 数组，处理起来末尾要加一个 `\0`。在 Python 中，`str` 是一个极其强大的对象。

### 1. 字符与编码的“转换器”

Python 3 内部字符默认使用 **Unicode** 编码：

* **获取整数表示：** `ord('中')` 得到 `20013`。
* **转换为字符：** `chr(20013)` 得到 `'中'`。
* **十六进制直接写：** `'\u4e2d\u6587'` 就是 `'中文'`。

### 2. `str` 与 `bytes`

这是很多初学者容易混淆的地方：

* **`str` (字符串)：** 是在代码里看到的文本，是抽象的“字符”。
* **`bytes` (字节流)：** 是数据在网络上传输或存在硬盘上的真实样子，长这样：`b'ABC'`。

* **编码 (Encode)：** 逻辑->现实。`'中文'.encode('utf-8')`
* **解码 (Decode)：** 现实->逻辑。`b'\xe4\xb8\xad'.decode('utf-8')`

> 永远坚持使用 **UTF-8** 编码。为了确保 Python 文件能正确处理中文，记得在文件开头写上：
> ```python
> #!/usr/bin/env python3
> # -*- coding: utf-8 -*-
> ```

## 二、 数据类型

### 1. 整数与浮点数

* **整数 (int)：** Python 的整数是**无限精度**的，再大的数也不会溢出（这和 C 完全不同！）。
* **浮点数 (float)：** 类似 C 的 `double`。可以用科学计数法：`1.2e9` 表示 。

### 2. 除法

这是 C 程序员最容易跌坑的地方：

* **精确除法 `/`：** `10 / 3` 的结果是 `3.3333333333333335`。即使整除，结果也是浮点数：`4 / 2` 得到 `2.0`。
* **地板除 `//`：** 类似于 C 的整数除法，直接丢弃小数部分。`10 // 3` 得到 `3`。
* **取余 `%`：** `10 % 3` 得到 `1`。

### 3. 布尔值与空值

* **布尔 (bool)：** `True` 和 `False`（注意首字母大写）。
* **逻辑运算：** 不是 `&&`, `||`, `!`，而是 `and`, `or`, `not`。
* **空值：** `None` == `NULL`。

## 三、 列表 (List)

### 1. 特点

* **有序集合**，通过索引访问。
* **类型不固定**：一个列表里可以同时放整数、字符串，甚至另一个列表。

### 2. 基本操作

```python
l = ['ZJU', 'BME']
l.append('Python')     # 追加：['ZJU', 'BME', 'Python']
l.insert(1, 'Great')   # 插入：['ZJU', 'Great', 'BME', 'Python']
l.pop(1)               # 删除索引1的元素
l[0] = 'Zhejiang Univ' # 替换

```
| 操作类型 | 方法/语法 | 说明 | 示例 |
|---------|----------|------|------|
| **长度** | `len()` | 获得 list 元素的个数 | `len(l)` |
| **访问** | `l[index]` | 用索引访问元素，索引从 0 开始。访问索引越界会报 `IndexError` 错误 | `l[0]`、`l[1]`、`l[-1]` |
| **追加** | `append()` | 在列表末尾添加元素 | `l.append('zhang')` |
| **插入** | `insert()` | 在指定位置插入元素 | `l.insert(1, 'que')` |
| **删除** | `pop()` | 删除指定索引的元素 | `l.pop(1)` |
| **替换** | 直接赋值 | 通过索引直接赋值替换元素 | `l[1] = 'xxx'` |
| **二维数组** | 嵌套 list | list 元素也可以是另一个 list | `s = ['python', 'java', ['asp', 'php'], 'scheme']` |

> **倒序访问：** Python 支持负数索引。`l[-1]` 表示最后一个元素，`l[-2]` 表示倒数第二个。

## 四、 元组 (Tuple)

`tuple` 和 `list` 唯一的区别是：**一旦初始化就不能修改**。它用小括号 `()` 定义。

```python
t = (1, 2)

```
它没有 append(), insert() 这样的方法。
**为什么要用元组？**

1. **安全：** 因为不可变，所以不会被程序无意中修改。
2. **性能：** 比列表稍微快一点点。

**避坑指南：**

* 定义只有一个元素的元组，必须加逗号：`t = (1,)`。如果不加逗号，`t = (1)` 会被 Python 认为只是一个带括号的数字 `1`。
* **“可变的”元组：** 如果元组里放了一个列表，这个列表内部是可以改的。但元组本身指向的那个列表对象并没变。
* `eg: t = ('a', 'b', ['A', 'B']) # 其中 ['A', 'B'] 元素是可被修改的`

## 五、循环

Python 的循环有两种：

### 1. for...in 循环

依次把 **list** 或 **tuple** 中的每个元素迭代出来。

**eg:**
```python
names = ['Michael', 'Bob', 'Tracy']
for name in names:
    print(name)
# 输出：
# Michael
# Bob
# Tracy
```

#### range() 函数

生成一个整数序列，通过 `list()` 函数可以转换为 **list**。

**eg:**
```python
list(range(5))  # [0, 1, 2, 3, 4]
list(range(1, 5))  # [1, 2, 3, 4]
list(range(0, 10, 2))  # [0, 2, 4, 6, 8]

for i in range(3):
    print(i)
# 输出：
# 0
# 1
# 2
```

### 2. while 循环

**eg:**
```python
n = 0
while n < 5:
    print(n)
    n = n + 1
# 输出：
# 0
# 1
# 2
# 3
# 4
```

#### break 语句

可以提前退出循环。

**eg:**
```python
n = 0
while n < 10:
    if n == 5:
        break
    print(n)
    n = n + 1
# 输出：
# 0
# 1
# 2
# 3
# 4
```

#### continue 语句

跳过当前的这次循环，直接开始下一次循环。

**eg:**
```python
n = 0
while n < 5:
    n = n + 1
    if n == 3:
        continue
    print(n)
# 输出：
# 1
# 2
# 4
# 5
```

## 六、字典 (dict)

**dict** 全称 **dictionary**，在其他语言中也称为 **map**。**dict** 使用**键-值（key-value）**存储，具有极快的查找速度。

### 声明

使用大括号 `{}` 声明。

**eg:**
```python
d = {'Michael': 95, 'Bob': 75, 'Tracy': 85}
```

### 访问

通过 **key** 访问对应的 **value**。

**eg:**
```python
d['Michael']  # 95
```

**注意**：如果 **key** 不存在，会报 `KeyError` 错误。

### in 函数

判断 **key** 是否存在。

**eg:**
```python
'Thomas' in d  # False
'Michael' in d  # True
```

### 访问方式 2：get() 方法

通过 **dict** 提供的 `get()` 方法，如果 **key** 不存在，可以返回 `None`，或者自己指定的 **value**。

**eg:**
```python
d.get('Thomas', -1)  # -1（key 不存在时返回默认值）
d.get('Michael')  # 95
d.get('Thomas')  # None（注意：返回 None 的时候 Python 的交互式命令行不显示结果）
```

### 删除 key

使用 `pop(key)` 方法，对应的 **value** 也会从 **dict** 中删除。

**eg:**
```python
d.pop('Bob')  # 75（返回被删除的 value）
# d 现在是 {'Michael': 95, 'Tracy': 85}
```

**重要提示**：**dict** 内部存放的顺序和 **key** 放入的顺序是**没有关系的**。也就是说，**dict 是乱序的**（在 Python 3.7+ 中，dict 保持了插入顺序，但不应依赖顺序）。

## 七、集合 (set)

**set** 和 **dict** 类似，也是一组 **key** 的集合，但**不存储 value**。由于 **key** 不能重复，所以，在 **set** 中，**没有重复的 key**。

**应用场景**：
- **去重**：快速去除列表中的重复元素
- **集合运算**：交集、并集、差集等
- **成员测试**：快速判断元素是否在集合中（比 list 快）

### 声明

创建一个 **set**，需要提供一个 **list** 作为输入集合。

**eg:**
```python
s = set([1, 2, 3])  # {1, 2, 3}
s = set([1, 1, 2, 2, 3])  # {1, 2, 3}（自动去重）
```

### 添加元素

使用 `add(key)` 方法可以添加元素到 **set**。

**eg:**
```python
s.add(4)  # {1, 2, 3, 4}
```

### 删除元素

使用 `remove(key)` 方法可以删除元素。

**eg:**
```python
s.remove(4)  # {1, 2, 3}
```

### set 之间的操作

支持交集、并集等集合运算。

**eg:**
```python
s1 = set([1, 2, 3])
s2 = set([2, 3, 4])

s1 & s2  # {2, 3}（交集）
s1 | s2  # {1, 2, 3, 4}（并集）
s1 - s2  # {1}（差集）
s1 ^ s2  # {1, 4}（对称差集）
```

**注意**：**set** 和 **dict** 的唯一区别仅在于没有存储对应的 **value**，但是，**set** 的原理和 **dict** 一样。

## 八、再议不可变对象

**str** 是不变对象（**immutable object**）。

**eg:**
```python
a = 'abc'
a.replace('a', 'A')  # 'Abc'（返回新字符串）
a  # 'abc'（原字符串不变）
```

**原理**：`a.replace('a', 'A')` 时创建了一个新的 **str**，所以 **a** 不被影响。

**可变对象 vs 不可变对象**：
- **可变对象**：**list**、**dict**、**set** 等，可以直接修改
- **不可变对象**：**str**、**tuple**、**int**、**float** 等，不能直接修改，只能创建新对象
