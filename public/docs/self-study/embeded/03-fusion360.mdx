# 3D建模与Fusion360

> Fusion360是Autodesk的免费3D建模软件，功能强大，适合机械设计和3D打印。

## Fusion360基础

### 软件获取

- **个人版**：免费（功能受限，但足够使用）
- **教育版**：免费（需要教育邮箱）
- **商业版**：付费

### 界面介绍

主要工作区：
- **浏览器**：管理设计历史和时间轴
- **视图立方体**：快速切换视角
- **工具栏**：常用工具
- **时间轴**：参数化设计历史

### 基础操作

#### 1. 草图（Sketch）

- 创建平面草图
- 绘制基本图形（线、圆、矩形等）
- 添加约束（尺寸、几何关系）
- 完成草图

#### 2. 特征（Feature）

- **拉伸（Extrude）**：将草图拉伸成3D
- **旋转（Revolve）**：绕轴旋转草图
- **扫掠（Sweep）**：沿路径扫掠
- **放样（Loft）**：连接多个轮廓

#### 3. 装配（Assembly）

- 创建组件
- 添加约束（配合、对齐等）
- 运动仿真

## 四足机器人结构设计

### 设计思路

1. **模块化设计**：每条腿独立设计，便于修改
2. **参数化**：关键尺寸设为参数，便于调整
3. **轻量化**：在保证强度的前提下减轻重量
4. **易装配**：考虑3D打印和装配的便利性

### 主要部件

#### 1. 身体（Body）

- 安装主控板
- 安装电池
- 连接四条腿
- 预留传感器安装位置

**设计要点**：
- 重心尽量低
- 内部空间合理利用
- 预留走线空间

#### 2. 大腿（Thigh）

- 连接身体和膝关节
- 安装第一个舵机（髋关节）

#### 3. 小腿（Shank）

- 连接膝关节和踝关节
- 安装第二个舵机（膝关节）

#### 4. 足部（Foot）

- 接触地面
- 安装第三个舵机（踝关节）
- 考虑防滑设计

#### 5. 舵机支架

- 固定舵机
- 连接相邻部件
- 考虑散热

### 设计参数

关键尺寸（需要根据实际舵机尺寸调整）：

```
身体尺寸：约 80mm × 60mm × 40mm
大腿长度：约 60mm
小腿长度：约 80mm
足部尺寸：约 30mm × 20mm
总高度：约 120mm（站立时）
总长度：约 150mm
```

### 设计步骤

1. **创建参数表**
   - 定义关键尺寸为参数
   - 便于后续调整

2. **设计单条腿**
   - 从足部开始
   - 逐步向上设计
   - 考虑舵机安装

3. **设计身体**
   - 根据PCB尺寸设计
   - 预留安装孔

4. **装配验证**
   - 在Fusion360中装配
   - 检查干涉
   - 运动仿真

5. **优化设计**
   - 减轻重量（挖空、减薄）
   - 增加强度（加强筋）
   - 优化外观

## 3D打印

### 打印设置

#### 材料选择

- **PLA**：易打印，强度一般，适合原型
- **PETG**：强度好，韧性好，推荐
- **ABS**：强度高，但需要加热床
- **TPU**：柔性材料，适合特殊部件

**推荐**：PETG（平衡了强度和打印难度）

#### 打印参数

```
层高：0.2mm（平衡速度和质量）
填充：20-30%（根据部件要求）
支撑：需要时添加
打印速度：50-60mm/s
温度：根据材料调整
```

#### 后处理

- 去除支撑
- 打磨表面
- 钻孔（如需要）
- 装配测试

### 打印注意事项

1. **方向选择**：考虑强度和打印质量
2. **支撑添加**：悬垂部分需要支撑
3. **尺寸补偿**：考虑打印收缩
4. **公差设计**：预留装配间隙

## 装配

### 装配顺序

1. 安装舵机到支架
2. 组装单条腿
3. 将腿安装到身体
4. 安装PCB和电池
5. 走线和固定
6. 最终调试

### 装配工具

- 螺丝刀（M2、M3）
- 内六角扳手
- 热熔胶枪（固定线材）
- 扎带（整理线材）

## 常见问题

### Q: 打印出来的零件尺寸不对？
A: 
- 检查打印机校准
- 考虑材料收缩率（PLA约0.2%）
- 调整模型尺寸补偿

### Q: 零件强度不够？
A:
- 增加壁厚
- 提高填充率
- 改变打印方向
- 使用更强材料

### Q: 装配时太紧或太松？
A:
- 调整孔的公差（通常+0.2mm）
- 使用不同打印参数测试
